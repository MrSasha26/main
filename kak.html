Как сделать свой набор с новыми словами?
Первым делом вам надо скачать "ПИЛОРАМУ" с официального сайта вместе с готовым набором с доп. словами (https://mrsasha26.github.io/main/pilorama.html). Затем надо открыть файл pocket.json. Помните, что название менять нельзя, иначе "ПИЛОРАМА" при попытке импорта набора выведет ошибку. Так же вам надо уметь разбираться в русском языке.

Первая строка отвечает за версию "ПИЛОРАМЫ", для которой создан набор. Если версия набора не будет совпадать с версией "ПИЛОРАМЫ", то при попытке импорта выведется ошибка. Поэтому версии должны совпадать.

Вторая строка отвечает за подлежащия в ед. ч.

Третяя строка отвечает за подлежащия во мн. ч.

Четвёртая строка отвечает за сказуемые несовершенного вида, которым не нужно дополнение (например, замерзал и т.п.).

Пятая строка отвечает за сказуемые несовершенного вида, которым нужно дополнение (например, поднимал, создавал и т.п.).

Шестая строка отвечает за сказуемые, которые используйтся для того, чтобы как-то назвать предмет (например, MrSasha26 - имба и т.п.).

Седьмая строка отвечает за сказуемые совершенного вида, которым не нужно дополнение (например, вышел, вошёл и т.п.).

Восьмая строка отвечает за сказуемые совершенного вида, которым нужно дополнение (например, растопил (что?), приготовил (что?) и т.п.).

Девятая строка отвечает за сказуемые настоящего времени, которым нужно дополнение (например, делают (что?), готовят (что?) и т.п.).

Десятая строка отвечает за сказуемые прошедшего времени несовершенного вида, которым нужно дополнение (например, делают (что?), готовят (что?) и т.п.).
